# ⚖️ India Legal AI

An advanced AI platform for Indian legal judgment analysis and search. Designed for lawyers, researchers, and citizens to navigate the complex landscape of Indian law with conversational intelligence.

## 🚀 Features

- **Semantic Search**: Understands legal concepts, not just keywords.
- **AI Synthesis**: Condense long judgments into actionable briefs.
- **Verified Data**: Direct citations to official court records.
- **Premium UI**: Modern, fast, and accessible design system.

## 🏢 Project Structure

```text
backend/        # Python (Flask/FastAPI)
  ├── app.py           # API Entry Point
  ├── ai_engine/       # AI Logic & RAG
  └── routes/          # API Endpoints

frontend/       # React + Vite + Tailwind CSS
  ├── src/             # Application Logic
  └── components/      # UI Components

data/           # Local Data Storage
  └── judgments/       # Raw Legal Texts

docs/           # Documentation
  └── architecture.md  # System Design
```

## 🛠️ Getting Started

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `python app.py`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
